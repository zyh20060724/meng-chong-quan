# 萌宠圈 · Pet Pals

一款手绘卡通风格的宠物社交应用，类似朋友圈。每位用户选择独特的宠物形象（猫、狗、兔子、仓鼠、蛇等）作为社区中的身份代表。

## 功能概览

| 页面 | 功能 |
|------|------|
| **主页** | 浏览公开动态、点赞、评论、发表情 |
| **好友** | 好友动态、私聊、添加好友、邀请小游戏 |
| **发文** | 发布文字/图片动态，设置公开/好友/私密 |
| **宠物** | 喂食、玩耍、你问我答、家具商店、装扮、图鉴 |
| **我的** | 编辑昵称/宠物名/简介、查看动态、设置 |

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 技术栈

- React 18 + TypeScript
- Vite
- React Router
- localStorage 本地数据持久化（演示版）

## 说明

当前为前端演示版本，数据保存在浏览器本地。如需上线多人真实社交，可接入后端 API 与数据库。
